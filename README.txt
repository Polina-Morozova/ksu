#Проект для практики в КГУ.


#Git


Учебная технологическая (проектно-технологическая) практика проводится в следующей форме: непрерывно, путем выделения в календарном учебном графике непрерывного периода учебного времени для проведения всех видов практик, предусмотренных ОПОП.


Цель учебной технологической (проектно-технологической) практики -  закрепление теоретических и практических знаний, полученных при изучении базовых дисциплин, развитие навыков поиска, критического анализа и синтеза информации, применения системного профессиональной деятельности.
