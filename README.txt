#Проект для практики в КГУ.


#Git

Индивидуальное задание на период практики:
-	Разработать мероприятия обеспечивающие самостоятельное изучение документации Git, консоль Git-bash основные команды, хостинг GitHub, клиент GitKraken, реализовать самостоятельные подходы в области освоения материала (УК-6); 
-	Инсталлировать и настроить Git в рабочей среде пользователя, зарегистрироваться на хостинге GitHub, настроить клиент GitKraken (ОПК-3);
-	Создание и клонирование репозитория, изменение истории проекта, публикация истории изменений (ОПК-1, ОПК-3);
-	Работа на хостинге GitHub: управление учетной записью, программный интерфейс GitHub (ОПК-6);
-	Разместить в Git-репозитории проект, синхронизировать работу локального и внешнего репозиториев (ОПК-7);
-	Внести изменения в проект партнера используя ресурсы хостинга GitHub (ОПК-8);
-	Оформить отчетную документацию, по итогам изучения Git (УК-6).
